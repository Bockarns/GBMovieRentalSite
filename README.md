# GBMovieRentalSite
This project is still under development. A complete Readme will be available when finished.

*Project description*
A fullstack-project that simulates a movierental site, where users can browes through diffrent movies and rent.

*Tech*
-Frontend (Bootstrap based atm during development)
-Backend (ASP.NET MVC, EF Core)
-Database (MS SQL LocalDB)

*Functions*
-Show movies
-Filter movies (Not implemented yet)
-Search movies (Not implemented yet)
-Add movies (Admin)
-Remove movies (Admin)
-Edit movies (Admin)
-Login (User and Admin)
-Rent movie (Not implemented yet)
